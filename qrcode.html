XSS Cookie Access Demo

This simulates how an attacker could access cookies if a site is vulnerable to XSS.

Try it safely:

Add this to the URL (this demo is safe - it won't actually execute):

?data=<script>alert(document.cookie)</script>

How to Protect Against This:

  • Always sanitize user input with libraries like DOMPurify
  • Use Content Security Policy (CSP) headers
  • Set cookies with HttpOnly flag (blocks JavaScript access)
  • Use frameworks that auto-escape (React, Vue, Angular)
