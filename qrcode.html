How It Works

If a website takes the data parameter from the URL and inserts it directly into the page without sanitization, malicious JavaScript could execute.

For example, if the URL was:

https://project-matter.github.io/connectedhomeip/qrcode.html?data=<script>alert('XSS')</script>

And the page did:

// Vulnerable code example (DON'T DO THIS)
document.getElementById('qrData').innerHTML = urlParams.get('data');
